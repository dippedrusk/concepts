# Abstract Concepts in NLP

This repository contains conceptual and critique papers about various abstract concepts in NLP from reasoning and bias to interpretability and stereotypes, for my class.
I welcome contributions via PR or email, and will try to keep this up to date as more relevant work comes out.

## General Readings

Reading papers
- [How to Read a Paper](https://web.stanford.edu/class/cs114/reading-keshav.pdf) - S. Keshav (2001)
- [How to Read a Technical Paper](https://www.cs.jhu.edu/~jason/advice/how-to-read-a-paper.html) - Jason Eisner (2009)
- [Ten simple rules for reading a scientific paper](https://journals.plos.org/ploscompbiol/article?id=10.1371/journal.pcbi.1008032) - Maureen A. Carey, Kevin L. Steiner, William A. Petri J (2020)
- [Reading a Technical Paper](https://jofrhwld.github.io/teaching/courses/2022_lin517/resources/reading/) - Josef Fruehwald (2022)
